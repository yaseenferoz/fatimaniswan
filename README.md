# Jamia Fatima Niswan website

A responsive bilingual (English/Urdu) school website built with HTML, CSS and JavaScript.

Pages included:

- Home (`index.html`)
- About (`about.html`)
- Learning (`learning.html`)
- Campus (`campus.html`)
- Contact and exact location (`contact.html`)

## Run locally

From this folder, start any static file server, for example:

```powershell
python -m http.server 4173
```

Then open `http://127.0.0.1:4173` in a browser. Use `?lang=ur` to open the Urdu version directly.

The chosen language is retained while navigating between pages.
